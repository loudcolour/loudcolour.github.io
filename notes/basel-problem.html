Basel 문제

Basel 문제는 Riemann의 ζ\zetaζ 함수 ζ(s)=∑k=1∞1ks\zeta(s) = \sum_{k=1}^{\infty}\frac{1}{k^s}ζ(s)=∑k=1∞​ks1​ 에 대하여 ζ(2)\zeta(2)ζ(2) 의 값을 정확하게 닫힌 꼴로 구하는 문제이다.

이중적분과의 관계를 이용하는 풀이

단위 사각형 □\square□

과 이중적분

간단히 쓰기 위해, □\square□ 는 R2\R^2R2 의 부분집합인 [0,1]×[0,1][0,1]\times[0,1][0,1]×[0,1] 을 의미하는 것으로 하자. 즉, ∬□f dxdy\iint_{\square}f,dxdy∬□​fdxdy 는 ∫01∫01f dxdy\int_{0}^{1}\int_{0}^{1}f,dxdy∫01​∫01​fdxdy 와 같은 의미이다. 이 부분집합 내에서의 이중적분인, ∬□dxdy1−xy\iint_{\square}\frac{dxdy}{1-xy}∬□​1−xydxdy​ 가 ζ\zetaζ 함수형의 급수꼴로 나타내어지는 것을 보이자.

먼저, ∣x∣<1|x|<1∣x∣<1 의 상황에서 11−x=∑k=0∞xk\frac{1}{1-x} = \sum_{k=0}^{\infty}x^k1−x1​=∑k=0∞​xk 꼴의 멱급수 전개를 적용하여

∬□dxdy1−xy=∬□∑k=0∞xkyk dxdy=∑k=0∞∬□xkyk dxdy=∑k=0∞1(k+1)2=∑k=1∞1k2\iint_{\square}\frac{dxdy}{1-xy} = \iint_{\square}\sum_{k=0}^{\infty}x^ky^k,dxdy = \sum_{k=0}^{\infty}\iint_{\square}x^ky^k,dxdy = \sum_{k=0}^{\infty}\frac{1}{(k+1)^2} = \sum_{k=1}^{\infty}\frac{1}{k^2}∬□​1−xydxdy​=∬□​k=0∑∞​xkykdxdy=k=0∑∞​∬□​xkykdxdy=k=0∑∞​(k+1)21​=k=1∑∞​k21​

∬□dxdy1−xy=ζ(2)\iint_{\square}\frac{dxdy}{1-xy} =\zeta(2)∬□​1−xydxdy​=ζ(2) 임을 보였다. 이와는 별개로 Dilogarithm 함수 Li2(x)\text{Li}_{2}(x)Li2​(x) 를 이용하는 방법도 고려할 수 있다. Li2(x)\text{Li}_2(x)Li2​(x) 의 정의는 다음과 같다.

Li2(x)=∫x0ln⁡(1−t)t dt\text{Li}2(x) = \int{x}^{0}\frac{\ln (1-t)}{t},dtLi2​(x)=∫x0​tln(1−t)​dt

이 때, Li2(x)\text{Li}2(x)Li2​(x) 의 정의에 따라, ∬□dxdy1−xy\iint{\square}\frac{dxdy}{1-xy}∬□​1−xydxdy​ 를 yyy 에 대해서 적분한 꼴이 −∫01ln⁡(1−x)x dx-\int_{0}^{1}\frac{\ln(1-x)}{x},dx−∫01​xln(1−x)​dx 이므로, ∬□dxdy1−xy=Li2(1)\iint_{\square}\frac{dxdy}{1-xy} = \text{Li}2(1)∬□​1−xydxdy​=Li2​(1) 이 성립함을 알 수 있다. ∣t∣<1|t|<1∣t∣<1 인 경우에 대해 −ln⁡(1−x)-\ln(1-x)−ln(1−x) 를 Taylor 전개한 결과는 −ln⁡(1−t)=∑n=1∞tnn-\ln(1-t) = \sum{n=1}^{\infty}\frac{t^n}{n}−ln(1−t)=∑n=1∞​ntn​ 이므로, Li2(x)\text{Li}_2(x)Li2​(x) 의 급수 전개는

Li2(x)=−∫0xln⁡(1−t)t dt=∑n=1∞∫0xtn−1n dt=∑n=1∞xnn2\text{Li}2(x) = -\int{0}^{x}\frac{\ln (1-t)}{t},dt =\sum_{n=1}^{\infty}\int_{0}^{x}\frac{t^{n-1}}{n},dt =\sum_{n=1}^{\infty}\frac{x^n}{n^2}Li2​(x)=−∫0x​tln(1−t)​dt=n=1∑∞​∫0x​ntn−1​dt=n=1∑∞​n2xn​

임을 알 수 있다. 결론적으로, ζ(2)=Li2(1)\zeta(2) = \text{Li}2(1)ζ(2)=Li2​(1) 가 성립하고, ∬□dxdy1−xy=ζ(2)\iint{\square}\frac{dxdy}{1-xy} =\zeta(2)∬□​1−xydxdy​=ζ(2) 임을 보였다.

다른 적분: 닫힌 값을 구할 수 있는 적분

위에서는 ∬□dxdy1−xy\iint_{\square}\frac{dxdy}{1-xy}∬□​1−xydxdy​ 꼴의 적분이 ζ\zetaζ 함수의 형태로 나타내어짐을 보였다. 이번에는 ζ(2)=π26\zeta(2)=\frac{\pi^2}{6}ζ(2)=6π2​ 이라는 결과를 비슷하지만 다른 형태의 적분을 도입하여 직접 도출할 것이다. 이를 위해, 실제로 닫힌 값을 구할 수 있고, ζ(2)\zeta(2)ζ(2) 를 이용하여 결과를 나타낼 수 있는 적분을 고려한다. 이 경우, ∬□dxdy1−x2y2\iint_{\square}\frac{dxdy}{1-x^2y^2}∬□​1−x2y2dxdy​ 를 적당한 적분으로 볼 수 있다.

먼저, 이 적분을 ζ(2)\zeta(2)ζ(2) 를 이용하여 나타내어자. 전의 장에서 다루었던 적분 ∬□dxdy1−xy\iint_{\square}\frac{dxdy}{1-xy}∬□​1−xydxdy​ 를 이용하면, 21−x2y2=11−xy+11+xy\frac{2}{1-x^2y^2} = \frac{1}{1-xy}+\frac{1}{1+xy}1−x2y22​=1−xy1​+1+xy1​

가 성립하므로,

∬□dxdy1−x2y2=12(∬□dxdy1−xy+∬□dxdy1+xy)\iint_{\square}\frac{dxdy}{1-x^2y^2} = \frac{1}{2}\left(\iint_{\square}\frac{dxdy}{1-xy} +\iint_{\square}\frac{dxdy}{1+xy}\right)∬□​1−x2y2dxdy​=21​(∬□​1−xydxdy​+∬□​1+xydxdy​)

또한, 2xy1−x2y2=11−xy−11+xy\frac{2xy}{1-x^2y^2}=\frac{1}{1-xy}-\frac{1}{1+xy}1−x2y22xy​=1−xy1​−1+xy1​

가 성립하므로, u=x2u=x^2u=x2 , v=y2v=y^2v=y2 으로 치환하면,

∬□dxdy1−xy−∬□dxdy1+xy=∬□2xy dxdy1−x2y2=12∬□dudv1−uv=12∬□dxdy1−xy\iint_{\square}\frac{dxdy}{1-xy} - \iint_{\square}\frac{dxdy}{1+xy} = \iint_{\square}\frac{2xy,dxdy}{1-x^2y^2} = \frac{1}{2}\iint_{\square}\frac{dudv}{1-uv} = \frac{1}{2}\iint_{\square}\frac{dxdy}{1-xy}∬□​1−xydxdy​−∬□​1+xydxdy​=∬□​1−x2y22xydxdy​=21​∬□​1−uvdudv​=21​∬□​1−xydxdy​

다시말해,

12∬□dxdy1−xy=∬□dxdy1+xy\frac{1}{2}\iint_{\square}\frac{dxdy}{1-xy} = \iint_{\square}\frac{dxdy}{1+xy}21​∬□​1−xydxdy​=∬□​1+xydxdy​

이므로, 이 결과를 대입하면,

∬□dxdy1−x2y2=12(∬□dxdy1−xy+∬□dxdy1+xy)=34∬□dxdy1−xy=34ζ(2)\iint_{\square}\frac{dxdy}{1-x^2y^2} = \frac{1}{2}\left(\iint_{\square}\frac{dxdy}{1-xy} +\iint_{\square}\frac{dxdy}{1+xy}\right) = \frac{3}{4}\iint_{\square}\frac{dxdy}{1-xy} = \frac{3}{4}\zeta(2)∬□​1−x2y2dxdy​=21​(∬□​1−xydxdy​+∬□​1+xydxdy​)=43​∬□​1−xydxdy​=43​ζ(2)

이다. 즉, 이 적분의 결과는 ζ(2)\zeta(2)ζ(2) 에 유리수를 곱한 값으로 나타난다.

그렇다면 이제 이 적분을 직접 계산함으로써 닫힌 값인 34ζ(2)=π28\frac{3}{4}\zeta(2)=\frac{\pi^2}{8}43​ζ(2)=8π2​ 로 나타남을 보이자. ∬□dxdy1−xy\iint_{\square}\frac{dxdy}{1-xy}∬□​1−xydxdy​ 을 계산하기 위해 특수한 Jacobian을 도입하자. x=sin⁡ucos⁡vx=\frac{\sin u}{\cos v}x=cosvsinu​ , y=sin⁡vcos⁡uy=\frac{\sin v}{\cos u}y=cosusinv​ 로 두면,

dxdy=∣∂(x,y)∂(u,v)∣ dudv=∣cos⁡ucos⁡vsin⁡u⋅sin⁡vcos⁡2vsin⁡u⋅sin⁡vcos⁡2vcos⁡vcos⁡u∣ dudv=∣1−tan⁡2u⋅tan⁡2v∣ dudvdxdy = \left|\frac{\partial(x,y)}{\partial(u,v)}\right| ,dudv = \begin{vmatrix} \frac{\cos u}{\cos v} & \frac{\sin u\cdot\sin v}{\cos^2 v}\ \frac{\sin u\cdot\sin v}{\cos^2 v}& \frac{\cos v}{\cos u} \end{vmatrix},dudv = |1-\tan^2 u\cdot\tan^2 v|,dudvdxdy=∣∣∣∣∣​∂(u,v)∂(x,y)​∣∣∣∣∣​dudv=∣∣∣∣∣​cosvcosu​cos2vsinu⋅sinv​​cos2vsinu⋅sinv​cosucosv​​∣∣∣∣∣​dudv=∣1−tan2u⋅tan2v∣dudv

이다. 적분 범위를 새로 지정하기 위해 변환 사상 ϕ:(x,y)↦(u,v)\phi:(x,y)\mapsto(u,v)ϕ:(x,y)↦(u,v) 에 의한 상 ϕ(□)\phi(\square)ϕ(□) 을 살펴보자. ϕ−1\phi^{-1}ϕ−1 가 삼각함수로 구성된 사상임을 고려하면, ϕ\phiϕ 가 well-definend이기 위해서는 ϕ(□)⊂[0,π2]×[0,π2]\phi(\square)\subset[0,\frac{\pi}{2}]\times[0,\frac{\pi}{2}]ϕ(□)⊂[0,2π​]×[0,2π​] 을 만족하는 것으로 충분하다. 또한, 0≤xy=tan⁡u⋅tan⁡v≤10\leq xy=\tan u\cdot\tan v\leq 10≤xy=tanu⋅tanv≤1 이므로, tan⁡u≤1tan⁡v=tan⁡(π2−v)\tan u\leq\frac{1}{\tan v}=\tan\left(\frac{\pi}{2}-v\right)tanu≤tanv1​=tan(2π​−v) , tan⁡\tantan 가 단조증가함수임에 따라 u+v≤π2u+v\leq\frac{\pi}{2}u+v≤2π​ 가 성립한다. 결론적으로 새 적분범위는 ϕ(□)={(u,v) ∣ 0≤u,v≤π2, u+v≤π2}\phi(\square) = \left{(u,v),|,0\leq u,v\leq\frac{\pi}{2},,u+v\leq\frac{\pi}{2}\right}ϕ(□)={(u,v)∣0≤u,v≤2π​,u+v≤2π​} 임을 알 수 있다. 적분을 계산하면,

∬□dxdy1−x2y2=∬ϕ(□)∣1−x2y2∣ dudv1−x2y2=∬ϕ(□)dudv=π28\iint_{\square}\frac{dxdy}{1-x^2y^2} =\iint_{\phi(\square)}\frac{|1-x^2y^2|,dudv}{1-x^2y^2} =\iint_{\phi(\square)}dudv =\frac{\pi^2}{8}∬□​1−x2y2dxdy​=∬ϕ(□)​1−x2y2∣1−x2y2∣dudv​=∬ϕ(□)​dudv=8π2​

위에서 구한 적분의 결과에 따라, 34ζ(2)=π28\frac{3}{4}\zeta(2) = \frac{\pi^2}{8}43​ζ(2)=8π2​ , 즉 ζ(2)=π26\zeta(2)=\frac{\pi^2}{6}ζ(2)=6π2​ 이다.

같은 결과에 도달하는 다른 방법: 급수를 조작하기

∬□dxdy1−x2y2=∫01tanh⁡−1(y)y dy=∑k=0∞∫01y2k2k+1 dy=∑k=0∞1(2k+1)2\iint_{\square}\frac{dxdy}{1-x^2y^2} = \int_{0}^{1}\frac{\tanh^{-1}(y)}{y},dy = \sum_{k=0}^{\infty}\int_{0}^{1}\frac{y^{2k}}{2k+1},dy = \sum_{k=0}^{\infty}\frac{1}{(2k+1)^2}∬□​1−x2y2dxdy​=∫01​ytanh−1(y)​dy=k=0∑∞​∫01​2k+1y2k​dy=k=0∑∞​(2k+1)21​
